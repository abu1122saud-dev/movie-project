🎬 Movie List Manager (Django Project)
📝 Project Description
This is a web application built using the Django framework. The project is designed to manage and display movie lists, showcasing the implementation of backend logic, URL routing, and dynamic view rendering. It serves as a practical application of core web development concepts in the Django ecosystem.

🛠 Technologies Used
- Python: The core programming language.
- Django Framework: The high-level Python web framework for rapid development.
SQLite: Light-weight database used for data storage.
-HTML: For structuring the web pages.
- Git & GitHub: For version control and project hosting.
